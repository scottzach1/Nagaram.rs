# Nagaram.rs

A rust implementation of Nagaram inspired by [scottzach1/Nagaram](https://github.com/scottzach1/Nagaram)

## Usage

```bash
cargo run "nagaram"
#------------------
    Finished dev [unoptimized + debuginfo] target(s) in 0.00s
     Running `target/debug/nagaram nagaram`
- "rag am an"
- "rag an am"
- "man agar"
- "agar man"
- "an rag am"
- "an am rag"
- "anagram"
- "am rag an"
- "am an rag"
```
